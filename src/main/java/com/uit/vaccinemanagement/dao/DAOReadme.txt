Each DAO class provides CRUD operations for its corresponding model using JDBC and DBConnection.
- BenhDAO: for Benh
- NguoiDungDAO: for NguoiDung
- NhaSanXuatDAO: for NhaSanXuat
- VaccineDAO: for Vaccine
- ThongTinMuaDAO: for ThongTinMua
- TiemChungDAO: for TiemChung

You can use these DAOs in your controllers to interact with MySQL data.
